# Python Study Assistant

This is a simple Python program that helps students manage their study tasks.  
It allows adding tasks, sorting them, calculating total study hours, finding the longest task, removing duplicates, and generating motivational messages.

## Features
- Add tasks
- View task list
- Total study hours
- Highest time task
- Remove duplicate tasks
- Sort by priority
- Motivational messages
